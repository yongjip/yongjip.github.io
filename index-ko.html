AI를 활용해 데이터 문제를 단순하고 확장 가능한 시스템으로 만듭니다.

실제 비즈니스 운영을 움직이는 데이터 플랫폼, API, 내부 시스템을 만듭니다. 제 관심사는 화려한 기술 시연보다 실용적인 실행입니다. AI 코딩 에이전트를 활용해 모호한 데이터 문제를 내부 제품, 반복 가능한 워크플로우, 그리고 더 빠른 의사결정을 돕는 도구로 바꿉니다.

소개

데이터 엔지니어링, 분석, 프로덕트의 교차점에서 일합니다.

제 기반은 데이터 시스템이지만, AI 코딩 에이전트를 활용하면서 더 넓은 범위를 직접 구현하게 됐습니다. 단순히 대시보드나 파이프라인을 넘어서, 비즈니스 문제가 end-to-end 해법을 요구할 때는 경량 서비스와 내부 도구까지 만듭니다.

  • 소규모 팀에서 100개 이상의 mart 테이블과 50개 이상의 대시보드 운영
  • Athena, Lambda, Glue, EMR Serverless, DynamoDB 기반의 serverless 선호
  • 수작업 분석을 줄이고 의사결정 속도를 높이는 시스템에 집중

철학

복잡한 시스템보다 잘 설계된 단순한 시스템이 더 자주 이깁니다.

대부분의 고임팩트 데이터 문제는 거대한 플랫폼 재구축이 필요하지 않습니다. 명확한 구조, 신뢰할 수 있는 검증, 그리고 실제로 유지 가능한 워크플로우가 더 중요합니다. 저는 설명하기 쉽고, 반복 개선이 가능하며, 실제 운영을 버틸 수 있는 시스템을 선호합니다.

복잡한 문제도 잘 설계된 단순한 시스템으로 해결할 수 있는 경우가 많습니다.
